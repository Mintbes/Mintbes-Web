Modern, high-performance website for the **Mintbes Harmony ONE Validator**. Built with React, Vite, Tailwind CSS 4, and Framer Motion.

## Features
- **Secure Staking Dashboard**: Integration links for easy delegation.
- **Mintbes Arcade**: Play-to-earn Web3 games ($ONE rewards).
- **Harmony Gallery**: AI-generated media celebrating the ecosystem.
- **Responsive Design**: Optimized for all devices.
- **Micro-animations**: Smooth transitions with Framer Motion.

## Tech Stack
- **React 18**
- **Vite** (Build Tool)
- **Tailwind CSS 4** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`

## Deployment
This site is designed to be hosted on GitHub Pages or Vercel.
